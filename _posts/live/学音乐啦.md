## 认识口琴

![20260326165850.png](../../assets/学音乐啦/20260326165850.png)

记住: **1 3 5吹** , **2 4 6 7吸**

曲谱的符号解释：- 的意思是拉长音，吸------ 吹------

关键：**吹吸对节奏，移动对吹吸**

## 第一支曲子~ 玛莉有只小羊羔

### 玛莉有只小羊羔谱

```js
3 2 1 2 3 3 3 -
2 2 2 - 3 3 3 -
3 2 1 2 3 3 3 -
2 2 3 2 1 - 0 0
```
