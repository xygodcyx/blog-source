{% paranote "summary" "作者以夏日午后的场景开篇，营造怀旧氛围。" %}
今日，窗外梧桐叶的影子落在桌面上，{% marginalia "收音机" "此处指1987年松下RF-2200型短波收音机" %}里传来沙沙的调频声。那个下午没有手机通知{% fn %}，只有磁带转动的声音和慢慢融化的冰块。
{% endparanote %}

{% paranote "citation" "此段据作者1987年日记整理，原文尚有'读着狄更斯'一句。" "" "left" %}
在维多利亚时代，人们通过书信和报纸来传递思想，那种缓慢的交流方式如今已经成为一种奢侈。
{% endparanote %}

{% emphasis %}偶然性{% endemphasis %}本身就是一种珍贵的体验。
